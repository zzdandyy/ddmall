# ddmall -微服务架构商城项目

基于 spring 2.4.3 构建的微服务项目，spring cloud 使用目前的 2020.0.1 版本

现在基本上都没有最新版本的项目，新版本的 spring boot 很多服务框架都用不了，所以想尝试一下，业务模块划分尽量贴合互联网公司的架构体系

## 版本更新

1.0.0 基础框架搭建

- 由于 ereuka 不再维护，使用 conusl 代替，版本 1.9.3
- 导入 sleuth+zipkin 进行链路追踪
